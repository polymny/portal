Polymny Studio

L'estudi web per a formadors que creen, editen i gestionen vídeos d'educació

A partir d'una presentació existent (libre office, powerpoint, beamer, etc.), proporcioneu les vostres diapositives en PDF i enregistreu un vídeo educatiu per als vostres alumnes, estudiants, clients o companys.

Programari lliure

Polymny Studio és programari lliure i es pot utilitzar de forma gratuïta, només cal que creeu un compte per començar a crear els vostres vídeos.

{% set image = resize_image(path="../static/img/portability.png", height=64, op="fit_height", format="webp") %}
Portabilitat

Polymny Studio funciona amb Windows, MacOS i Linux, un simple navegador, Firefox o Chrome, n'hi ha prou per crear els teus vídeos.

{% set image = resize_image(path="../static/img/pdf.png", height=64, op="fit_height", format="webp") %}
Diapositives PDF

Utilitzeu la vostra eina preferida per preparar el vostre contingut, exportar en format PDF i registrar-vos a Polymny Studio.

{% set image = resize_image(path="../static/img/play.png", height=64, op="fit_height", format="webp") %}
Producció automàtica

Baixeu el vostre vídeo final en format MP4 o publiqueu-lo directament al servidor de streaming de Polymny Studio.

Compartint càpsules

No es requereix cap experiència tècnica, ni edició, ni compressió, ni manipulació digital de vídeos. Les càpsules s'emmagatzemen i es publiquen automàticament en línia en un servidor de vídeo. S'hi accedeix mitjançant un enllaç web (url) que es pot compartir per correu electrònic o missatge de text, a les xarxes socials o en qualsevol plataforma de formació (moodle, 360, etc.). La figura de l'esquerra mostra vídeos de polimnies compartits a Moodle.

{% set image = resize_image(path="../static/img/moodle.png", width=500, op="fit_width", format="webp") %}

Registre fàcil

A diferència del programari de gravació de pantalla, Polymny orienta i facilita l'enregistrament de comentaris de vídeo (amb la càmera web) o comentaris d'àudio (amb el micròfon de l'ordinador). El registre es fa simplement des d'un navegador web.

{% set image = resize_image(path="../static/img/recording.png", width=500, op="fit_width", format="webp") %}

4 passos

El protocol Polymny es basa en 4 passos: preparar (utilitzar o no utilitzar el teleprompter), filmar (possiblement limitar-se al so), produir (col·locar els mitjans segons diferents patrons) i publicar (aconseguir l'enllaç per compartir sense cap esforç tècnic). ).

{% set image = resize_image(path="../static/img/bigpicture.png", width=500, op="fit_width", format="webp") %}

Gestionar les càpsules

No es malgasta cap minut d'enregistrament, cada esforç es pot modificar i reutilitzar més tard: com al cinema, pots multiplicar les “presas” per satisfer el director (tu!). Ja no enregistres la teva pantalla sinó que gestiones els teus projectes de càpsula de vídeo, d'un any a l'altre, d'una classe a una altra, d'una versió inicial a la següent millora! La interfície de Polymny mostra 3 projectes d'usuari: 3 seqüències educatives cadascuna de les quals inclou diverses càpsules de vídeo, algunes de les quals estan en preparació i encara no s'han finalitzat. La gestió de cada càpsula com a projecte permet la intervenció d'enginyers pedagògics o comunicadors en la fase de preparació dels recursos pedagògics: alineació pedagògica, ajust de la durada del missatge o la claredat del vídeo.

{% set image = resize_image(path="../static/img/management.png", width=500, op="fit_width", format="webp") %}

Vídeos addicionals

Entre les funcions populars, Polymny us permet inserir, en lloc d'una diapositiva, qualsevol vídeo extern (captures dels vostres escrits a tauletes per a professors, vídeos o captures de pantalla des d'un telèfon intel·ligent, clips lliures de drets importats d'Internet). La il·lustració següent mostra un jingle de vídeo (una animació) inserit com a crèdits d'un vídeo d'entrenament. Polymny és utilitzat amb èxit pels formadors en l'àmbit públic (universitats, instituts, MOOC FUN) i privat (organitzacions de formació professional).

{% set image = resize_image(path="../static/img/addresource.png", width=500, op="fit_width", format="webp") %}

Funcions avançades

Les funcions avançades estan disponibles com a serveis addicionals i opcionals. Si la teva càpsula, ja preparada i prototipada al web, mereix una versió professional, l'equip de Polymny t'ofereix:

  • serveis de producció en estudis audiovisuals virtuals (clau, teclejat/matting per deeplearning, generació gràfica de fons i primer pla dinàmics, punters, plantilles d'edició personalitzades, sistema de recomanació per a l'edició gramatical),
  • serveis de producció en estudis físics accessibles mitjançant reserva (tecla i tecles de pantalla verda, múltiples càmeres i plans, so d'alta qualitat, escenaris de diàleg multimicròfon, pre/postproducció).
